Moderní rozvržení webu

Flexibilní box model a Mřížkové rozvržení

Úvod

Flexbox a CSS Grid jsou základní pilíře moderního webdesignu. Umožňují vytvářet responzivní weby, které se automaticky přizpůsobí velikosti obrazovky bez nutnosti složitých výpočtů.

Flexbox (1D rozvržení)

Navržen pro uspořádání prvků v jednom směru – buď jako řádek, nebo sloupec.

.container { display: flex; justify-content: center; }

Využití: Navigační menu, centrování ikon, řazení karet produktů.

CSS Grid (2D rozvržení)

Systém mřížky, který pracuje ve dvou rozměrech zároveň (sloupce i řádky).

.grid { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 10px; }

Využití: Celková struktura stránky, komplexní galerie, magazínové layouty.

Srovnání

Layout Dimenze Hlavní zaměření
Flexbox 1D (řada nebo sloupec) Obsah prvků
Grid 2D (řady i sloupce) Celková struktura

Závěr

Obě techniky se skvěle doplňují. Často se používá Grid pro hlavní kostru webu a Flexbox pro detaily uvnitř prvků.
